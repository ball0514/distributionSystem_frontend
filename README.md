# 📦 配送系統

## 關於新立企業社內部作業系統

本專案旨在推動新立企業社的作業流程自動化，核心目標如下：
● 多維統計整合： 透過系統化架構，一鍵生成多種統計報表，解決單一資料需重複計算之問題。
● 效率極大化： 精簡人工運算流程，將重複性行政作業自動化，大幅節省人力工時。
● 資訊零時差： 廢除傳統紙本作業，達成即時狀態更新與數據同步，確保資訊決策的即時性與準確性。

## 後續規劃

確定資料格式之後，預計導入轉換成JSON格式的AI自動化應用。
